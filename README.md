# Student-Management-System
This project demonstrates a simple Student Management System developed using the Python programming language. The system helps users perform simple operations such as adding, searching, updating, deleting, and storing student records.

  The system is a console-based application that allows users to manage a list of students. Each student has personal and academic characteristics, and users can manage the system via a menu.

The key features in the menu for managing the system are as follows:

•	Adding a new student  
•	Finding an existing student  
•	Showing all students  
•	Showing students by age range  
•	Updating student information 
•	Deleting a student 
•	Saving student data to a file 
•	Loading student data from a file
